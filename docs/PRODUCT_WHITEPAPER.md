# PinSeeker Web - 产品白皮书 & 操作流程 (SOP)

## 1. 产品定位
**PinSeeker** 是一款基于 Web 的高精度高尔夫策略与分析工具。它不仅仅是一个测距仪，而是球手的**智能电子球童 (Smart Electronic Caddie)**。

### 核心价值主张
*   **可视化风险管理**：通过散布圆 (Dispersion Circles) 帮助球手理解击球落点概率，避开障碍区。
*   **私有化地图数据**：解决偏远/私人球场无数据痛点，提供专业级地图编辑器。
*   **零门槛使用**：基于 PWA 技术，无需下载 App Store 安装包，通过浏览器即可获得原生级体验。

| 用户类型 | 特征描述 | 核心需求 |
| :--- | :--- | :--- |
| **策略型球手** | 差点 10-20，注重成绩管理 | 不再盲目进攻，需要知道“哪里是安全的失误区域”。 |
| **数据发烧友 (Geeks)** | 清楚自己每支杆的 Carry 和偏差数据 | 需要将练习场数据应用到下场实战中，进行科学决策。 |

| **教练/教学机构** | 需要复盘工具 | 需要在地图上画线、标记，向学员解释球场策略。 |

## 3. 核心功能亮点

### A. 击球散布分析 (Dispersion Analysis) 🔥
不同于传统 APP 只显示一个“点”，PinSeeker 根据用户设置的 **Carry (飞行距离)**、**Side Error (左右偏差)** 和 **Depth Error (纵深偏差)**，在地图上渲染出一个蓝色的概率椭圆。
> **价值**：直观展示90%的落点范围。如果椭圆覆盖了水障碍，球手就知道必须调整瞄准方向，直到椭圆完全处于安全区。

### B. DIY 球场编辑器 (Course Editor) 🛠️
内置专业级 GIS 地图编辑器，支持 22 级深度变焦。
> **价值**：用户可以拖拽 Tee 台和果岭位置，自定义 Par 信息。真正做到“我的主场我做主”。

### C. 交互式策略板 (Interactive Strategy) ✏️
提供画笔、文字备注、旗帜标记工具，以及可视化的风向罗盘。
> **价值**：像职业比赛的码数本一样做笔记，标记果岭速度、隐蔽沙坑等。

### D. 智能测距仪 (Advanced Rangefinder) 🎯
支持 "My Location" 瞬时定位。
> **价值**：站在球道任意位置，一键测量到沙坑前沿（Layup）或旗杆的精确距离。

## 4. 竞品差异化对比

| 功能维度 | PinSeeker (本项目) | 主流竞品 (18Birdies/Arccos) |
| :--- | :--- | :--- |
| **测距逻辑** | **散布概率模型 (由此决定瞄准点)** | 简单的点对点直线距离 |
| **地图精度** | **用户可编辑 (Level 22 极清)** | 官方数据库 (更新慢，通用精度) |
| **风向辅助** | **可视化罗盘 (直观显示侧/逆风)** | 简单的箭头或文字描述 |
| **使用门槛** | **Web App (即开即用，隐私安全)** | 需要下载几百 MB 安装包，强制注册 |
| **策略工具** | **支持手绘线、橡皮擦、备注** | 仅提供简单的目标点移动 |

## 5. 下场全流程 SOP (User Journey)

### 第一阶段：赛前准备 (Pre-Round)
1.  **球包管理**: 进入 `Settings > Club Management`。调整主力球杆的平均飞行距离（Carry）和偏差范围。
2.  **选择球场**: 点击 `START NEW ROUND`。APP 基于 GPS 推荐最近球场。若无数据，使用 `Add New Course` 快速描点。

### 第二阶段：发球台 (The Tee Box)
1.  **设置风向**: 点击右侧 `Wind` 图标，转动罗盘设置风向。
2.  **策略制定**: 观察 Driver 落点散布圈。如果右侧有水，拖动瞄准滑块向左，直至蓝圈避开障碍。
3.  **击球**: 按照 APP 指示方向击球。

### 第三阶段：球道行进 (Fairway)
1.  **记录位置**: 到达球位后，点击底部 **GPS 按钮**。APP 记录上一杆落点并切换至下一杆。
2.  **攻果岭**:
    *   开启 **测距模式 (Ruler)**，点击 "MY LOC" 确认当前到旗杆距离。
    *   若果岭硬，使用 **画笔 (Pen)** 标记备注。

### 第四阶段：果岭结算 (Green)
1.  **记分**: 点击右上角旗帜图标，输入推杆数 (Putts) 和罚杆数。
2.  **保存**: 点击 `Save Score` 自动跳转下一洞。

### 第五阶段：赛后复盘 (Post-Round)
1.  **结束**: 点击 `Finish Round` 查看记分卡。
2.  **回放**: 进入 **Replay Mode**，查看每一杆的实际轨迹（实线）与计划散布（虚线），复盘失误原因。
